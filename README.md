# AI Customer Support Chatbot

AI-powered chatbot for customer support that provides instant, smart responses and improves user experience in web applications.

## Features

- AI-powered customer support
- User authentication and registration
- Email verification with OTP
- MongoDB integration
- Chatbot embed code support
- Responsive dashboard UI
- Gemini AI integration

## Tech Stack

- Next.js
- React
- Tailwind CSS
- Motion
- MongoDB
- Gemini AI
- Axios
- Scalekit SDK

## Getting Started

Install dependencies:

```bash
npm install 

```Run the development server:
npm run dev
